```
메타버스
- 서로 상호작용이 가능한 가상 세계
- 모바일처럼 하나의 패러다임이자 플랫폼
```

```
기업별 생각하는 메타버스의 필수 요소
- 위메이드 : 이코노미(실제와 상호작용 가능한), 다른 자아
- SKT : 커뮤니케이션, 다른 자아
- 더 샌드박스 코리아 : 콘텐츠, 이코노미, 경험, 문화
```

```
메타버스에서 가장 중요한 것은 또 다른 자아이다.
현실 세계에서 하지 못했던 것들을 가상 세계에서 경험함으로써 자신이 알지 못했던 자아의 실현이 가능해진다.
하지만, 다른 자아에 너무 몰입하면 여러가지 문제가 발생할 수 있다.
```

```
VR : 가상 현실
AR : 증강 현실
MR : AR + 두 공간이 서로의 공간을 인식하고 상호작용이 가능할 때
XR : 위 기술들을 통틀어 이르는 단어
```

```
디지털 트윈
- 현실 세계의 것을 디지털로 옮겨놓은 것
- 최근에는 단순히 옮긴 것 뿐만 아니라 SCADA, 스마트 팜 같은 실시간 위주로 발전
```
